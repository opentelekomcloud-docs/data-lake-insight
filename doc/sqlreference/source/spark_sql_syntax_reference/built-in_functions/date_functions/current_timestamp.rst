current_timestamp
=================

This function is used to return the current timestamp.

Syntax
------

.. code-block::

   current_timestamp()

Parameters
----------

None

Return Values
-------------

The return value is of the TIMESTAMP type.

Example Code
------------

The value **1692002816300** is returned.

.. code-block::

   select current_timestamp();
